I wanted to have a shorter CDN link for FontAwesome so i created a plugin to keep things tight.

  • create a font-awesome.rb file inside your _includes directory
  • build your CDN link
  • register a new font_awesome tag to Liquid
# _includes/font-awesome.rb

module Jekyll
  module FontAwesomePlugin
    class Tag < Liquid::Tag

      LINK_TEMPLATE = '<link rel="stylesheet" type="text/css" href="//maxcdn.bootstrapcdn.com/font-awesome/%s">'

      def render context
        LINK_TEMPLATE % @markup.strip.concat("/css/font-awesome.min.css")
      end
    end
  end
end

Liquid::Template.register_tag "font_awesome", Jekyll::FontAwesomePlugin::Tag
  • add {% font_awesome 4.2.0 %} inside your HTML’s <head> section and your done
